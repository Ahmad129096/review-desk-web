# ReviewDesk Web

Vite + React frontend for the ReviewDesk AI MVP.

## Setup

```bash
cd reviewdesk-web
pnpm install
cp .env.example .env
pnpm dev
```

The app expects the API at `VITE_API_URL`, defaulting to `http://localhost:4000`.

## MVP Screens

- Register and login
- Business setup
- Review entry and AI reply display
- Review status updates
- Follow-up task list
- Stripe checkout buttons
